ASL - Translator

ASL Translator is a mobile app that leverages machine learning and image recognition to translate ASL hand signs to plain text. Working as a full stack developer on this project I was able to expand on my knowledge of machine learning, mobile development, and AWS services.

The biggest hurdle for ASL Translator was configuring and training an image recognition model. My team and I went through several versions of our model each time tweaking the training data set so that the model could recognize similar looking handsigns and handsigns in different environments. In the end we were able to achieve an average of 85% confidence by ensuring our training images were of good quality resolution and varied in lighting.

Trees: A Visualization

A web based visualization of four basic tree data structures using the p5.js library. Binary Search Tree, AVL Tree, Red Black Tree, and Min Heaps are covered. The inspiration for this project came from my Data Structures and Algorithms class where I was really intrigued to implement these trees on my own.
